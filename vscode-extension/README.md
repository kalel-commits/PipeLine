# PipelineAI: Local VS Code Intelligence

**Zero-Config ML Risk Predictor for your IDE.**

PipelineAI brings real-time CI/CD risk assessment directly to your primary workspace. No complex setup—just push-button intelligence.

### 🌟 Features
- **Real-time Gutter Highlights**: Instantly see risk profiles as you type.
- **Commit Guard**: Analyzes your local working directory and warns you of high-failure commits before you push.
- **Explainable Insights**: Brief SHAP values provided directly in the status bar.
- **Integrated AI Mentor**: Suggestions appear in your notifications for quick remediation.

### 🚀 Getting Started
1. Ensure the PipelineAI Backend is running locally on `port 8000`.
2. Open any repository in VS Code.
3. PipelineAI will automatically begin indexing your local commit history to provide context-aware predictions.
